۱. نصب و راه‌اندازی لوکال

پیش‌نیاز

  • Python 3.11 یا بالاتر
  • ترمینال PowerShell یا CMD

گام ۱ — رفتن به پوشه پروژه

cd C:\AmoozeshyarV03

گام ۲ — محیط مجازی

python -m venv .venv
.\.venv\Scripts\Activate.ps1

اگر PowerShell اجازه نداد (یک‌بار):

Set-ExecutionPolicy -Scope CurrentUser RemoteSigned

گام ۳ — نصب وابستگی‌ها

pip install -r requirements.txt

گام ۴ — دیتابیس

python manage.py makemigrations core
python manage.py migrate

گام ۵ — کاربر ادمین

python manage.py createsuperuser

۲. اجرای پروژه

python manage.py runserver
آدرسکاربرد
/admin/پنل مدیریت با تم سفارشی
/docs/setup-theme/همین راهنما
/docs/راهنمای قبلی پروژه
در config/settings.py اپ core باید قبل از django.contrib.admin در INSTALLED_APPS باشد؛ وگرنه تم اعمال نمی‌شود.

۴. تغییر ظاهر

۴.۱ — رنگ تم تیره

فایل core/static/amoozeshyar/admin.css → بلوک html[data-theme="dark"]

html[data-theme="dark"] {
  --ay-primary: #5eead4;
  --ay-bg: #05080f;
  --ay-surface: #101827;
  /* ... */
}

۴.۲ — رنگ تم روشن

همان فایل → بلوک html[data-theme="light"]

۴.۳ — گوشه‌های گردتر

:root {
  --ay-r-sm: 10px;
  --ay-r: 14px;
  --ay-r-lg: 18px;
  --ay-r-xl: 24px;
}

۴.۴ — تم پیش‌فرض

admin-theme.js → خط const DEFAULT = 'dark'; را به 'light' تغییر دهید.

۴.۵ — متن دکمه تم و هدر

  • دکمه: base_site.html و login.html
  • برچسب دکمه در JS: admin-theme.js تابع applyTheme
  • عنوان سایت: config/urls.py → admin.site.site_header
  • پیام خوش‌آمد: templates/admin/index.html

۴.۶ — افزودن مدل جدید در ادمین

فقط در core/admin.py ثبت کنید؛ ظاهر خودکار از CSS مشترک می‌آید. نیازی به CSS جدا per-model نیست.

۵. رفع border تیز پشت گوشه گرد

مشکل از دو لایه border است: کادر بیرونی گرد + کادر مربع داخلی Django. در admin.css این‌طور حل شده:

  • کارت‌ها به‌جای border از box-shadow: 0 0 0 1px ... استفاده می‌کنند (دنبال‌کنندهٔ radius)
  • داخل #changelist و fieldset borderهای اضافی حذف شده‌اند
  • بخش پایانی فایل: «رفع border مربع داخل عناصر گرد»

اگر جایی هنوز تیز بود، در همان بخش پایانی CSS سلکتور همان المنت را اضافه کنید:

/* مثال */
.my-problem-box {
  border: none !important;
  border-radius: 0 !important;
  box-shadow: none !important;
}

۶. چک‌لیست بعد از تغییر CSS/JS

  1. ذخیره فایل
  2. ری‌استارت runserver
  3. رفرش سخت مرورگر: Ctrl+F5
  4. در صورت تم قدیمی: پاک کردن ay-admin-theme از Local Storage
